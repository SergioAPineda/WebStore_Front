Project Overview:


Working in a group (up to 6 members) and utilizing your accumulated knowledge of the

MEAN Stack (MongoDB, ExpressJS and NodeJS with AngularJS) create a Web App from one of the

following type of projects (details below):

  • Sell Used Products Site
  • Another type of project previously approved.

Your Web App must be hosted on a live site that supports a Node server.

You will have core functionality requirements as well as specific criteria for your type of Web App.
